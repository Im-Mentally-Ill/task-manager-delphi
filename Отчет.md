# Отчет о разработке веб-приложения на Delphi

## 1. Цель работы

Разработать веб-приложение архитектуры клиент-сервер с использованием:
- Среда разработки: **Embarcadero Delphi 10.2**
- Веб-сервер: **Встроенный веб-сервер Indy**
- База данных: **PostgreSQL 18** (подготовлена, ожидает подключения)

## 2. Функциональные требования

Приложение должно отображать список сотрудников компании в виде HTML-таблицы с полями:
- ID сотрудника
- ФИО
- Отдел
- Должность
- Зарплата
- Дата приема на работу
- Email

## 📌 О проекте

Веб-приложение для отображения списка сотрудников компании, разработанное на **Embarcadero Delphi 10.2** с использованием технологии **WebBroker**.

Приложение реализует архитектуру клиент-сервер: веб-браузер отправляет HTTP-запросы к серверу, написанному на Delphi, который динамически генерирует HTML-страницу с данными о сотрудниках.

## 🚀 Быстрый старт

### Требования

- Windows 7/8/10/11
- Embarcadero Delphi 10.2 (или выше)
- PostgreSQL 18 (опционально, для работы с БД)
